# Contributing

Thank you for your interest in contributing to Portfolio Tracker API.

This project was created as a backend portfolio project, but contributions, suggestions, and improvements are welcome.

## Getting Started

1. Fork the repository.
2. Clone your fork.
3. Create a new branch.
4. Make your changes.
5. Run tests.
6. Open a pull request.

## Local Setup

Install the required tools:

- .NET SDK
- Docker Desktop
- Git

Start PostgreSQL:

```bash
docker compose up -d postgres
```
